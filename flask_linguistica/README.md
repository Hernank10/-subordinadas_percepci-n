# 📚 Verbos de Percepción y Predicados Estativos

Aplicación educativa interactiva desarrollada con Flask que explora las restricciones sintácticas entre verbos de percepción y predicados estativos en español.

## 🎯 Características

- **Teoría interactiva**: Explicaciones detalladas sobre predicados estativos vs. dinámicos
- **Ejemplos contrastivos**: Comparación entre estructuras correctas e incorrectas
- **Ejercicios prácticos**: 3 ejercicios interactivos con retroalimentación inmediata
- **Sistema de Flashcards**: 100 flashcards con diferentes categorías y niveles de dificultad
- **Seguimiento de progreso**: Estadísticas de ejercicios completados y flashcards vistas

## 🚀 Tecnologías utilizadas

- **Backend**: Flask (Python)
- **Frontend**: Bootstrap 5, JavaScript
- **Almacenamiento**: JSON para datos, Sesiones para progreso de usuario
